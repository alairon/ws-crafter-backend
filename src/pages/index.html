WS Crafter Backend Site Map

Welcome to the backend system for WS Crafter!
If you came for the actual website, please visit the general user page (currently WIP), or view the source code at GitHub.

API Directory

API variables are represented with curly brackets { }. All of the following links below that requires {set} are set up to use the Goblin Slayer Expansion, using the GBS identifier.
GET API calls will return a JSON to the API caller when possible.

Cards (GET)

api/cards - Shows a simplified list of all the cards available on the server, regardless of set.
api/cards/{set} - Shows all of the cards available from a specified set.
api/cards/{set}/character - Shows all of the character cards available from a specified set.
api/cards/{set}/event - Shows all of the event cards available from a specific set.
api/cards/{set}/climax - Shows all of the climax cards available from a specific set.
api/cards/{set}/{cardNo} - Searches the set for a card containing the card number, as defined as {cardNo}. For example, '01' will show all cards that contain '01' in the card's number.

Cards (POST)

api/cards - Inserts card information from the card data creation tool at /cards

Sets (GET)

api/sets - Retrieves a list of sets available on the database.
api/sets/{set} - Retrieves information about a particular set from the database.

Sets (POST)

api/sets/{set}/{key} - Updates the database based on the set specified. Content is generated through /api/cards POST API calls from the creation tool. Requires an unencrypted passkey defined as {key} to prevent simple abuse.

Server Tools

/cards - The Card Creation Tool allows users to generate a JSON file stored on the server to be inserted into the database. This will eventually be migrated to the client page.
